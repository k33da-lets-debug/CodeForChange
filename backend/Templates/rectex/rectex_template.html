    We got your image, see here how it got processed

    OCR tools are used for text detection. We are using OpenCV for image preprocessing and Google's tessaract engine to extract data from it. Here are some interesting image processing results which will explain how this works in lay mans terms. There are so many filters which can be used to extract data more accurately.

    01

    Conversion 1

    For start you have uploaded your file, then to process it we converted it into OpenCV's readable format. By default OpenCv uses BGR format so we converted image into RGB format.

    02

    Conversion 2

    As we have got RGB image that means we have more colour pixel data. It will be so much memory intensive to perform operations for just Text extraction. So we converted it to Grayscale image.

    03

    Conversion 3

    After Converting to Grayscale still some more filteration process is required. If you can see we have some watermarks and some lousy noise in document. To remove it we simpelly removed Alpha channels and converted into a Binary image i.e Black and White

    04

    Conversion 4

    Sometimes documents loose/gain some unnecessary thickness/thinnes. If image is thick, we can erode it and make it little thin. If image is thick we can dilate it and make it thin.

    05

    Conversion 5

    Sometimes document may contain certain patterns of noise like lines, circle, particles. This filter is designed to remove horizontal lines from our dataset.

    06

    Conversion 6

    As previous filter removes horizontal lines, this one removes verticle lines. However it should be noted that it won't work as accurate for some results. Sometimes its good to add more filterations for removing tricky noise.

    Recognised Text

    After doing some image preprocessing here kicks in our actual OCR engine. Google's Tessaract project is mainely focused on creating best OCR solutions in no time. It is actually written in C++ but it is made availabe for other languages too. As currently we have built this in python so we are using pytessaract module which is wrapper for Tessaract engine.
